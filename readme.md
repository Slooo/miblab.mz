## Install

* git clone https://rep.miblab.ru/miblab/mz mz
* cd mz
* composer install
* copy **.env.example** in created the file **.env** options **DB_USERNAME=username** & **DB_PASSWORD=password** 
* php artisan key:generate
* php artisan migrate
* php artisan db:seed

### server machine
* chmod 777 resources
* chmod 777 storage

### local machine
* php artisan serve

**Done, yep!**
